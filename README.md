
# OPH2025-edit

editting information for oph2025
## Setup

```bash
  1.Install dependencies
    bun install (recommend)
    yarn install
    npm install
  2.Patch oauth module
    bun patch-package (recommend)
    yarn patch-package
    npx patch-package
  3.Start server
    bun dev (recommend)
    yarn run dev
    npm run dev
```
